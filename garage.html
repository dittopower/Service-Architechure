Service:

DeamonGarage

Service Type

REST

Purpose

To check if the garage is available and make booking for the garage.

Functionality

OperationUsageWhat it doesConstraintsPreconditions
check /check/{date} Takes a given date and returns an xml response of 1 for available and 0 for not. HTTP GET {date} is a date string in the form "YYYY-MM-DD"
make /make/{date}/{id} Takes a given date and customer id, Books the garage for that time and returns a generic confirmation message. HTTP GET {date} is a date string in the form "YYYY-MM-DD"
{id} is a numerical customer id
